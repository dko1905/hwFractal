# HardWare accelerated Fractals (C-vulkan)
![C-vulkan CI](https://github.com/Dko1905/hwFractal/workflows/C-advanced%20CI/badge.svg?branch=c-vulkan)
## Compiling & running
### Dependencies
Vulkan & GLFW

### Compiling
```sh
$ meson builddir
$ cd builddir && ninja all
```

### Running
Run `./hwfractal`
